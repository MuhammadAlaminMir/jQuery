Promise(): -- It is an important method from Deferred object. this method returns an object with almost the same interface then the Deferred.

Promise() method is useful when you want to give to the calling API something to subscribe to but not the ability to resolve or reject the deferred

In jquery .ajax() method returns a promise, and it has three main methods:

  • done() - this is default callback for handling successful ajax operations
  • fail() - this is default callback for handling rejected operations
  • always() - this method executes regardlss of weather the operation is done or fall
