jQuery Utilities

    We have so many Utilities in jQuery

    Like:

  • $.each(s): It iterates over each element in the collection and performs a callback on that object

  • $.map(): It iterates over each element in the collection and performs a callback on that object, but it didn't change the main array its make new one

  • $.trim(): its remove unwanted space to a string

  • $.inArray: its check the given value Exist in array or not

  • $isFunction(): this method check that the given value is a function or not

  • $isPlainObject(): this method check that the given value is a Object or not

  • $isArray(): this method check that the given value is a Array or not
