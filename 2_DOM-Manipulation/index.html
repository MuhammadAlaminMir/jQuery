DOM Manipulation

One fo the most important aspects of jQuery, is DOM Manipulation

DOM Manipulation using standard Javascript can be pretty cumbersome, fortunately jQuery comes with a bunch of DOM related methods, making it all much easier for us to work with DOM.

jQuery DOM Manipulation is easier to:

>> Manipulate element's content
>>Add/remove DOM elements
>>Modify an existing element
>>Copy/clone an element
>>Add/remove/modify elements attributes
